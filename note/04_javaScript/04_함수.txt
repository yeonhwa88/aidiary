함수 종류:
익명함수, 선언함수, 매개변수와 리턴값, 가변인자함수,  내부함수, 자기호출함수, 콜백함수, 클로저함수, 내장함수
1. 익명함수 : 코드의 집합을 나타내는 자료형 변수 (익명함수 호출은 변수이름뒤에 ()를 붙여 호출 / 함수 오버로딩불가) 
     - var functionVar = function() { };  함수 표현식
     - var functionVar = 0 => {}; 화살표 함수
     - 화살표 함수 매개변수 지정 방법
	- () => {} 매개변수가 없을경우
	- x => {} 매개변수가 한 개인 경우 소괄호 생략 가능
	- (x, y) => {} 매개변수가 2개이상인 경우 소괄호 생략 불가
     - 함수 몸체 지정 방법
	- x => {return x*x} single line block  
	- x =x*x=>{} 위표현과 동일 함수 몸체가 한줄의 구문이면 중괄호 생략 가능
	- () => {return {a:1};}
	- () ({a : 1}); 위 표현과 동일 객체 반환시 소괄호 사용 
	- () = > { // multi line block
		const x = 10;  
		return x*x;
		} 
     - 익명함수 생성과 출력 : 이름이 없으므로 변수에 넣어서 사용
2. 선언적함수 : function 함수명 () { // 함수 선언식 }
3. 매개변수와 리턴값 : function 함수이름 (매개변수1, 매개변수2, 매개변수3){ 함수코드 return값 }
     - 매개변수 : 지정한 매개변수보다 많거나 적은 매개변수를 사용하는 것을 허용한다. 
                   원래 함수에 선언된 매개변수보다 많게 사용하면 일반적으로 추가된 매개변수는 무시한다. 적게 사용하면 지정하지 않은 매개변수는 undefined
     - Array() 함수 ; 빈 배열 생성
     - Array(number); 매개변수만큼의 크기를 가지는 배열 생성
     - Array(1, 2, 3, ... ); 매개변수를 배열로 생성
4. 가변인자함수 : 매개변수를 선언된 형태와 다르게 사용했을때 매개변수를 모두 활용할 수 있는 함수 매개변수의 개수가 변할수 있다
     - arhuments : 매개변수 배열
5. 내부 함수 : 함수 내부에 선언하는 함수 ( 함수의 충돌을 막기위해 내부함수를 사용하면 내부함수가 우선된다 )
6. 콜백함수 : 함수도 하나의 자료형이므로 매개변수에 넣을 수 있다 . 매개변수에 넣어 전달된 함수를 콜백함수라 한다.
7. 클로저함수 : 리턴된 함수
8. 내장함수 :  자바스크립트에 내장된 함수, 확인할수있는 도큐먼트 주소 ((https://www.w3schools.com/jsref, http://devdocs.io/javascript)
9 타이머함수 : 
     - setTimeout(function, millisecond) 일정시간 후 함수를 한번 실행
     - setInterval(function, millisecond) 일정시간마다 함수를 반복해서 실행
     - clearTimeout(id) setTimeout 중지
     - clearInterval(id) setInterval 중지
