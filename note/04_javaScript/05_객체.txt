객체 : 서로 연관된 변수와 함수를 그룹핑하고 이름을 붙인것
속성과 메서드 : 배열 내부에 있는 값은 요소 , 객체 내부에 있는 값은 속성.
자바스크립트는 요소와 속성을 구분하지 않음
객체 속성 중 함수형 자료형인 속성을 특별히 메서드라고 한다.
객체와 반복문 : 배열은 단순 for 반복문과 for-in반복문으로 요소에 접근 
		단순 for문은 불가 for-in반복문을 사용한다
객체 속성 추가 : 객체이름. 변수이름 = "값"
객체의 속성 제거 : delete(객체이름.변수이름)
