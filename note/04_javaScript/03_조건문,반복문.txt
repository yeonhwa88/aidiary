조건문 종류: if문, switch문, 삼항연산자는 자바와 같음
if문, switch문, 삼항연산자, 짧은조건문, 내장객체(document, date(현재날짜), math(상수))
반복문 종류:
배열, for in, for of, forEach함수, do~while문, while문 (do~while문, while문은 자바와 같음)
  - 배열 [] : 어떤 자료형도 배열의 요소가 될 수 있다.
     - push : 인덱스 제일 끝 인덱스 뒤에 추가
     - unshift(0) : 인덱스 0번방에 추가 기존방은 옆으로 밀림
     - pop : 마지막 인덱스 값을 리턴 하고 삭제 
  - for in : 인덱스 번호와 값을 출력
  - for of : 인덱스 값만 출력 (자바에서 확장 for문과 비슷함)
  - forEach : 2가지의 방법으로 (인덱스, 값), (값) 출력가능
