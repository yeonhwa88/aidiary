var : 변수 선언시 재선언 가능 전역변수로 사용
let : 변수 선언시 재선언 불가 지역변수로 사용
const : 상수 , 새로운 값 재할당 불가 (객체로 사용시 내부값 변경가능)
자료형 : 
숫자 자료형 : 정수와 실수 구분없이 사용, %(나머지 연산자) 사칙연산 가능
불 자료형 : 비교연산자, 논리연산자 (!부정연산자, &&논리곱연산자, ||논리합연산자)
undefined 자료형 : 변수를 선언하지 않았거나 선언만하고 초기화 하지않았을때. 
입출력: 자바스크립트에 내장된 대화상자를 이용한 인터페이스
   1) alert 함수: 경고 메세지를 전달하기 위한 대화상자 표시
   2) comfirm 함수 : 사용자의 확인을 받기 위한 대화상자 표시
       반환값 = comfirm(출력할 메세지);// 반환값이 true면 확인  false면 취소
   3) prompt 함수 : 사용자로 내용을 입력 받을수 있는 대화상자 표시
       반환값 = prompt(메세지, 초기값);
문자열 + 숫자 자료형 변환 :
   - Number() : 다른 자료형 숫자로 바꿀 때 . 일반 문자열을 숫자로 표시할수 없지만 강제로 숫자 자료형으로 바꿀시 NaN 출력
   - parseInt(string,[진수]),parseFloat(string) : 문자열을 정수와 실수로 변경. 문자열 타입의 숫자만 가능
   - String() : 다른 자료형을 문자열로 변경.
   - isNaN(숫자) : 주어진 데이터가 숫자면 false리턴 , 숫자가 아니면 true리턴
불자료형 변환 :
   - boolean() : boolean 자료형으로 변환. (false로 해석되는 값 : 0, "", undefined, NaN, null)
일치 연산자 
   - == : 양변의 값이 일치할 경우 true
   - === : 양변의 자료형과 값이 일치할 경우 true
