객체와 클래스 :
class 클래스명(상속받을 클래스명):
	변수나 함수들 
변수와 메소드 :
- 클래스변수 vs 인스턴스변수 vs  static변수
- 인스턴스메소드 vs static메소드 vs 클래스메소드
메소드 :
인스턴스메소드 : 첫번째 인자에 self를 꼭 명시. 객체를 통해서면 호출할 메소드 (객체를 호출할 수 있는 함수)
클래스 메소드 :  메소드위에 @classmethod 선언한 메소드 , 첫번째 인자에 cls(clz) 클래스를 명시하는 단어를 꼭 명시
static메소드 : 메소드위에 @staticmethod 선언한 메소드. 디폴트 인자를 넣지 않음
생성자(__init__)와 소멸자(__del__) : 
__init__(self,매개변수..) 객체가 생성될 때 자동 실행 생성시 필요한 코드 (인스턴스 변수 초기화)를 포함
__del__(self) : 객체가 소멸될 때 자동 실행 소멸시 필요한 코드를 포함 인스턴스 객체의 래퍼런스 카운트가 0일 될 때 실행
static 변수 : 인스턴스변수 (생성자를 통함), 클래스변수(클래스내선언), static변수(클래스 내에 선언한 변수 중 변수명앞에__ (under score 2개)를 붙임
