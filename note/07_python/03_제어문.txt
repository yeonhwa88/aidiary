if문 : 조건이참일때 특정코드 실행,
자바와 다른점 비교 :  ex. if (score>=0) & (score<=100) / if 0 >= score <= 100
for- in문 : 리스트, 튜플, 문자열 같은 반복 가능한 객체에서 요소를 하나씩 반복 실행, 주로 정해진 횟수만큼 반복할 때 사용
else : 나열가능한 자료를 다 소요하여 for 블록을 다 수행한 후 최종 한번만 실행할 문장 (선택사항) for문 안에 break가 있으면 else블럭 수행 안됨
while문 : 한번 이상 횟수가 정해지지 않은 문장을 반복할 때 사용
	ex. 무한 루프에 빠지는것 조심
중첩루프 : 루프 안에 또 다른 루프를 사용하는 구조, 2차원 배열 처리할때 사용 (반복문 안 반복문)
중첩루프 탈출 : for문 이후 break를 만나면 안쪽 for문만 빠져나오고 for문 전체적으로 빠져나오지 못할때 사용하는 방법
  1. 플래그 변수 이용 : flag = False 지정해두고 break를 만날때 True로 변환하여 for문을 전체적으로 빠져나오게 함
  2. Exception 이용 : Exception 상속받은 class를 만들어 try - except 사이에 강제 예외를 발생시켜 빠져나오게 함
