1. 변수 : 변하는 값을 저장, 자료를 일괄성 있게 사용하고 관리, 연산에 변수 이름을 사용 할 수 있다.
1.1 변수 선언 : 파이썬은 변수 선언을 위한 타입이 없고, 별도의 선언 과정도 없음
1.1 변수 할당 : 할당연산자를 이용해서 변수값을 할당하면 타입이 셋팅, 파이썬에서 모든 변수는 객체
1.2 변수 이름 규칙 : 문자로 시작, 문자, 숫자, 밑줄_ 변수명 작성 
  함수명은 변수이름으로 가능하나 사용 지양. 사용하면 int_식으로 _붙여서 사용 
  파이썬 예약어(if,while,...)는 변수이름 사용 불가
  한글 변수는 꼭 필요시에만 사용
1.3 변수에 값 할당 : 변수에 값을 저장, 할당연산자 사용 (=) 
1.4 id() : 객체의 주소 값을 출력
1.5 변수삭제 : del변수(현재 커널 변수를 삭제), 여러 변수를 동시에 삭제하려면 목록에 ,로 구분해 나열
1.6 다중 변수 선언 : 한 라인에 여러 변수를 선언해 사용 ex : a, b = 10, 20
	        변수에 값을 할당 작업은 동시 할당 ! (순차 X) 
1.7 도움말 : ex : help(print) print 도움말 찾아볼수있으며 , shift+tap 으로 사용가능
2. 화면 i/o : 화면 입력과 출력 
2.1 화면 입력 : input()  : 사용자로부터 값을 입력받기 위한 함수. 입력값은 문자열
	  받을 내용출력은 input('입력해주세요') 사용가능
2.2 화면  출력 : print()
  sep = ' ' default값 스페이스 한개
  end = '\n" default값 엔터
  file = sys.stdout default값 현재 터미널
3. 기본 자료형 : 
자료형(데이터타입(data type)) : 변수가 가져야 할 값의 크기, 형식, 범위 값의 유형을 정해놓고 사용, 어떤 값을 가질수 있는지에 대해 알려주는 속성, 
			값의 범위와 값을 저장하는 방식이 달라짐, 수행 할 수 있는 명령들이 달라질수 있음, 변수를 선언할때 자료형을 지정하지 않음
3.1 숫자형 : 정수(int), 실수(float), 복소수(complex)
정수 int : 숫자의 크기가 얼마가 되던지 모든 정수는 int 자로형으로 처리
실수 float : 부동소수점 float형으로 처리 정수형과 정수형의 나눗셈 연산은 실수형 float
복소수 complet : 실수와 허수의 함을 나타내는 수 체계 허수는 i가 아닌 j를 붙여 표현 (ex : c = 3 +2j, d=1j print(d**2)
3.2 문자형 : 단일문자와 문자열을 구분하지않음, 겹따옴표("내용") 와 홑따옴표('내용')로 묶어서 사용
여러 줄 문자 표현  : 따옴표3개(""" 내용 """) 또는 홑따옴표3개(''' 내용 ''') 를사용 이렇게 사용하게되면 줄바꿈도 그대로 적용.
