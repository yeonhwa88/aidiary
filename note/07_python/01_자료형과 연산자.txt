변수 선언 : 별도의 선언 타입과 과정이 없음
변수 할당 : 할당연산자를 이용해서 변수값을 할당하면 타입이 셋팅, 모든 변수는 객체
변수 이름 규칙 : 문자로 시작, 문자(특수문자는 _제외 전부 불가), 숫자, _로 변수명 작성, 함수명은 변수이름으로 가능하나 피한다. 사용해야 하면 int_식으로 _를 붙여서 사용
	         파이썬 예약어(if, while,...)는 변수이름 사용 불가, 한글 변수는 꼭 필요시만 사용
포맷팅 : 출력할 문자, 숫자, 날짜 데이터에 형식을 지정하는 것, 변수와 텍스트를 결합해서 보기 좋은 형식의 새로운 텍스트를 만들 수도 있음
{[인덱스]:[공백대체문자][정렬방법][자릿수][타입]}".format(변수명)
    인덱스 : format 함수안의 매개변수의 순서
    정렬방법 : < : 왼쪽정렬, > : 오른쪽정렬, ^ : 가운데정렬
    정수자릿수 : 출력할 자릿수를 지정. 변수내용 보다 작은 자릿수가 지정되면 자릿수 지정 무시
    실수자릿수 : 전체출력자리수.변수정에출력할자리수
    타입 : d(10진수), o(8진수), b(2진수), x(16진수), f(실수), s(문자)
문자열 사용 변수 : 문자열슬라이싱변수[from:to:by] / len(문자열변수)
- 문자변수.upper() : 대문자 / 문자변수.lower() : 소문자
- title() : 각 어절의 앞글자만 대문자 / capitalize() : 첫문자만 대문자 
- count('찾을문자열') : '찾을문자열'이 몇개 들어있는지 갯수 반환
- find('찾을문자열') : '찾을문자열'이 왼쪽부터 시작하여  몇번째에 있는지 index를 반환 (자바 : indexOf) / rfind('찾을문자열') : '찾을문자열'이 오른쪽부터 시작하여 몇번째에 있는지 index를 반환 ('찾을문자열이 없으면 -1을 반환)
- index('찾을문자열') : '찾을문자열'이 왼쪽부터 시작하여 몇번째 있는지 index를 반환 ('찾을문자열이 없으면 오류 반환)
- startswith('특정문자열') : '특정문자열'로 시작하는지 여부 반환 / endswith('특정문자열') : '특정문자열로'로 끝나는지 여부 반환
- strip() : 좌우 공백제거(trim) / rstrip() : 오른쪽 공백제거 (rtrim) / lstrip() : 왼쪽 공백제거 (ltrim)
- split() : 문자열을 공백이나 다른 문자 기준으로 나누어 리스트로 반환 / join() : 리스트로 받은 내용을 연결시킬때 사용
- isdigit() : 문자열이 숫자 문자열인지 여부 반환 / islower() : 문자열이 소문자인지 여부 반환 / isupper() : 문자열이 대문자인지 여부 반환
- replace(oldStr, newStr) : oldStr을 newStr로 변경
