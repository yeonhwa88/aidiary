예외처리 :
예외가 날 가능성이 있는 부분에 대해 미리 예상하고, 그에 대한 처리를 프로그래밍 하는 것  ( 좀 더 안정적으로 실행할 수 있도록 함)
	try :
		데이터 수집코드 ( 예외가 발생할 가능성이 있는 명령어들 ...)
	except :
		오류가 났을 경우 대신 실행할 코드 ( 예외가 발생했을 경우 실행할 명령어들 ...)
예외를 지정한 처리 :
	try : 
		예외가 발생할 수도 있는 문장들
	except 예외타입1 :
		해당 예외가 발생할 경우 실행할 문장
	except 예외타입2 :
		해당 예외가 발생할 경우 실행할 문장
예외인수(e)
	try :
		예외가 발생할 수도 있는 문장:
	except 예외타입 as e :
		print(e) 튜플형태로 Error메세지로 출력
raise : 강제 예외 발생
