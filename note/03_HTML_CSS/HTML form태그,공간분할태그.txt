form태그 :
사용자에게 입력 받는 공간을 의미,입력 양식을 만들때 사용하는 태그
  - action : 입력 데이터 전달 위치 지정
  - method : 입력 데이터 전달 방식 선택
     - get : 길이 제한이 있으며 url뒤에 입력한 데이터가 눈에 보임
     - post : 파라미터 길이 제한이 없으며 데이터 영역을 이용하여 데이터를 전송 데이터가 눈에 보이지않음.
  - button : 버튼을 생성
  - checkbox : 체크박스를 생성
  - file :  파일 입력 양식을 생성
  - hidden : 보이지 않게 숨김
  - image : 이미지 형태 생성
  - password : 비밀번호 입력 양식 생성
  - radio : 라디오 버튼을 생성
  - reset : 초기화 버튼 생성
  - submit : 제출 버튼 생성
  - text : 글자 입력 양식 생성
