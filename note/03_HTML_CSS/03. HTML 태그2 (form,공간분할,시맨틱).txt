1. form태그 : 사용자에게 입력 받는 공간을 의미, 입력 양식을 만들 때 사용하는 태그 - <form 속성1=속성값 속성2=속성값> - <input type="~" name="전달할 파라미터이름" >
  1) action : 입력 데이터의 전달 위치 지정
  2) method : 입력 데이터의 전달 방식 선택 - get : url에 포함되어 전송 - post : 입력한 데이터가 눈에 보이지 않게 전송
  3) button : 버튼 생성
  4) checkbox : 체크 박스 생성
  5) file : 파일 입력 양식 생성
  6) hidden : 보이지 않게 생성
  7) image : 이미지 형태 생성
  8) password : 비밀번호 입력 양식 생성
  9) radio : 라디오 버튼 생성
  10) reset : 초기화 버튼 생성
  11) submit : 제출 버튼 생성
  12) text : 글자 입력 양식 생성
  13) autofocus : 포커스 자동 설정
  14) placeholder : 각 필드에 대한 입력 형식 설명 제공
  15) required :  필수 입력 필드
  16) min,max : 숫자의 범위 설정
  17) pattern : 조건을 사용한 일반식 표현
  18) lable : 사용자 인터페이스를 높이기 위한 사용
2. 공간분할태그
  1) div : block 형식의 공간 분할 (개행포함)  2) span : inline 형식의 공간 분할
3. 시맨틱태그 :
사용자가 눈으로 볼 때 이것은 제목이고 이것은 본문이다 식으로 구분 가능
시맨틱 웹의 장점은 검색엔진이 보다 효율적으로 검색 할 수 있고 유지보수시 효율적
  1) header : 해더
  2) nav : 네비게이션
  3) aside : 사이드에 위치하는 공간
  4) section : 여러 중심 내용을 감싸는 공간
  5) article : 글자가 많이 들어가는 부분
  6) footer : 꼬리말
