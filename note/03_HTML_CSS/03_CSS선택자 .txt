선택자란 : HTML 문서 중 특정 부분을 선택하여, 해당 부분의 속성을 변경하는 목적으로 사용
1. 태그선택자 : 특정 태그를 선택하여 CSS속성을 적용.
2. 전체(*) 선택자; 페이지 초기화 : 문서 전체를 선택하여 적용
3. id(#)선택자 class(.)선택자 : 태그의 id와 class를 이용하여 CSS속성 설정
4. 혼합선택자 : tag, id, class 혼합하여 CSS에서 속성 설정
5. 속성선택자 / input[type='password'] / input[type^='te'] / input[name$='me']
     - [속성=값] : 속성 값이 특정값과 같은 경우를 찾아 적용
     - [속성^=값1] : 속성 값이 지정한 문자로 시작하는 속성값에 대해 적용
     - [속성$=값1] : 속성 값이 지정한 문자로 끝나는 속성값에 대해 적용
6. 후손선택자, 자손선택자(바로아랫단) / #wrap > #header 자손 / #wrap .logo 후손
7. 동위(형제)선택자(+하나, ~다) h3~div / h3+div 
8. 반응선택자(ex.마우스를 올릴때 반응) / ul > li:hover
  마우스를 올려 놓았을때 (hover상태)의 반응데 따른 속성 설정
9. 상태선택자(ex.input에 focus된 상태) / input:focus
  상태에 따라 CSS속성이 변화하는 설정 (input태그의 경우 focus가 맞혀졌을 때)
10. 구조선택자 / :nth-child(2n) 짝수 / :nth-child(2n+1) 홀수
  구조에 따라 CSS속성이 변화하는 설정
11. 문자선택자 / :nth-child(2):first-line
  문자(문자열)중에 선택적 CSS속성 설정
12. 링크선택자 / :after{content:'-' attr(href)}
  문서에 링크(href) 되어 있는 문자를 선택하여 옆에 링크를 같이 띄워주는 CSS속성 설정
13. 부정선택자 / :not(.class이름)
  나를 제외한 모든 태그에 CSS속성 설정
