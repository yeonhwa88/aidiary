i/o (입출력) 이란 ? :
입력 (Input)과 출력 (Output)을 뜻한다.
프로그램에게 입력하는것은 Input 이며 프로그램에서 출력하는 것을 Output이라 한다.
파일 입출력 API :
InputStream / OutputStream : 1byte단위 입/출력 API. 이미지, 동영상 등의 데이터에 주로 사용.
Reader / Writer : 2byte단위 입/출력 문자열에 주로 사용.
스트림(Stream) 이란 ? :
데이터를 운반 (입출력) 하는데 사용되는 연결통로. 연속적인 데이터 흐름을 물에 비유해서 붙여진 이름.
하나의 스트림으로 입출력을 동시에 수행할 수 없다.(단방향 통신)
입출력을 동시에 수행하려면 2개의 스트림이 필요하다.
BufferedReader, BufferedWriter 클래스 :
속도를 빨리 하기 위한 보조 스트림
