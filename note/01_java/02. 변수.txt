변수 선언 방법 :
1.자료형 변수명; - 변수 선언과 할당 분리
2. 자료형 변수명 = 값; - 변수 선언과 동시에 초기화 
3. 자료형 변수1, 변수2, ...; 동일 자료형의 변수를 다수 선언 
4. 자료형 변수1=값1, 변수2=값2,...; 동일자료형 변수를 선언과 초기화를 동시에 가능하다. 
타입변수 :
기본 데이터 타입 (메모리에 있는 실제값 = 변수 데이터 값 
참조변수 데이터 타입 (객체 데이터 타입) (메모리에 있는 실제값 = 변수가 저장된 주소)
기본 데이터 타입 : 
기본변수 논리형 boolean / 문자형 char / 정수형 byte, short, int, long / 실수형 float, double 있다
콘솔 출력 하는법 : 
println() : 개행포함 
printf() : 포맷 지정. 개행\n 미포함(기본변수 단축키 (%d 정수, %f 실수, %c 문자, %s 문자열, %b boolean)) 
print() : 개행이 미포함된 println
형변환 : 
묵시적 형변환(공간이 큰 변수에 공간이 작은 값을 할당) 
명시적 형변환(공간이 작은 변수에 공간이 큰 값을 할당 / 데이터 손실이 생길수 있다 )
