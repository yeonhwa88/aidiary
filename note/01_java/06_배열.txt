배열이란 : 동일한 데이터 타입의 값을 하나의 변수에 연속적으로 저장하는 자료구조이다.
기본 값 : 배열은 기본적으로 데이터 타입에 따른 기본 값으로 초기화된다.(예: int는 0, boolean은 false).
배열 선언 : 배열을 선언할 때 데이터 타입과 배열 크기를 명시한다.
초기화 : 배열을 선언과 동시에 초기화할 수 있다.
인덱스 : 배열 요소는 0부터 시작하는 인덱스로 접근.
길이 : 배열의 길이는 .length로 확인할 수 있다.
다차원 배열 : 배열 안에 배열을 넣어 2차원, 3차원 등 다차원 배열을 만들 수 있다.
배열 반복 : 반복문을 사용해 배열의 모든 요소에 접근할 수 있다.
