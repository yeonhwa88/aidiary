메소드란? 
특정 작업을 수행하는 코드 블록으로, 여러 번 호출할 수 있다.
메소드 선언: 
반환 타입, 메소드 이름, 매개변수를 포함해 선언.
메소드 활용: 
코드를 재사용하고, 가독성을 높이며, 유지보수를 쉽게 만들어주는 핵심 요소.
반환값: 
메소드는 작업 후 값을 반환할 수 있으며, 반환이 필요 없을 때는 void를 사용한다.
매개변수:
메소드는 매개변수를 받아 그 값을 이용해 작업을 수행할 수 있다.
메소드 호출: 
메소드를 호출할 때는 메소드 이름과 필요한 매개변수를 전달한다.
오버로딩(Overloading)
같은 이름의 메소드를 여러 개 정의할 수 있으며, 매개변수의 타입이나 개수로 구분한다.
접근 제어자: 
메소드 앞에 public, private 등을 붙여 접근 범위를 제어할 수 있다.
static 메소드: 
클래스에 속하며, 객체 생성 없이 호출할 수 있는 메소드.
