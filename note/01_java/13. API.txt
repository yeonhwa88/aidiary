API란 ? :
편리하게 이용할수 있는 기능들을 정의 해둔 클래스.
Object 클래스의 메소드들
public String toString() 객체의 문자열을 반환. (내부적으로 동작하는 메소드중 하나).
public boolean equals (Object obj) 
public int hashCode() 자바에서 객체를 식별하는 정수 값인 해시 코드를 반환 (다량의 데이터를 저장, 검색하는 해싱 기법에 사용).
public Class getClass() 객체의 클래스를 이름에 Class형으로 반환.
Wrapper 클래스의 이해 :
기초데이터를 객체 데이터로 변환하는 클래스.
