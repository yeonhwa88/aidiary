조건문이란 : 프로그램 흐름을 제어하는 문법으로, 특정 조건이 참(true)일 때만 코드 블록을 실행한다.
if문 : 가장 기본적인 조건문으로, 조건이 참이면 중괄호 내부의 코드를 실행한다.
else문 : if문의 조건이 거짓(false)일 때 실행할 코드를 정의한다. 
중첩 if문: if문 안에 또 다른 if문을 사용할 수 있습니다. 조건이 복잡할 때 사용된다.
삼항 연산자: 조건문을 간단하게 표현할 때 사용합니다. 여러 조건을 체크하고 싶을 때 사용하며, 첫 조건이 거짓일 경우 다음 조건을 확인한다.
switch문: 여러 값을 비교해야 할 때 사용하는 조건문이다. case로 각 값에 따라 다른 동작을 수행한다.
switch의 break: 각 case의 마지막에 break를 써서 조건에 맞는 동작 후 나가도록 한다.
default: switch의문에서 모든 case가 해당되지 않을 때 실행할 코드를 정의한다.
