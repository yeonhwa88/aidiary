Sequence 정의 : 순차적인 번호를 자동으로 생성하는 객체로 테이블과 독립적으로 생성 및 저장.
(1) 시퀀스에서 생성되는 번호는 유일하기 때문에 primary key 생성 사용
(2) 여러 테이블 공유 가능
(3) 테이블과 관계없이 생성, 저장.
문법 :
CREATE SEQUENCE 시퀀스이름
START WITH 	-- 1부터 시작
INCREMENT BY 1   -- 1씩 증가
MAXVALUE 	-- 최대값
MINVALUE 	-- 최소값
NOCACHE 	-- 시퀀스값 생성 방지
NOCYCLE  	-- 만들어진 시퀀스 수가 가득 찼을때 처음으로 돌아가지 못하게 방지.
